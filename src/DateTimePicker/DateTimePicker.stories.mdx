# DateTimePicker
With `MDX` we can define a story for `DateTimePicker` right in the middle of our
markdown documentation.

## Simple 
<Preview>
	<Story name='Simple'>
    {
      ()=>{
        const [open, setOpen] = useState(false);
        const [state, setstate] = useState(new Date());
        const setDate = (date) => {
          setstate(date._d);
          setOpen(false);
        };
        return (
          <DateTimePicker open={open} value={state} onChange={(date) => setDate(date)} onFocus={() => setOpen(true)} />
        );
      }
    }
  </Story>
</Preview>
