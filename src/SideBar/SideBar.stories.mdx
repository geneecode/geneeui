# SideBar
With `MDX` we can define a story for `SideBar` right in the middle of our
markdown documentation.

## Simple 
<Preview>
  <Story name='Simple'>
    {
      ()=>{
        const [open, setOpen] = useState(false);
        const toggleDrawer = useCallback(
        () => setOpen(!open),
        [open]
        );
    return(
        <SideBar
          onToggleDrawer={toggleDrawer}
          open={open}
        />
        )
      }
    }
  </Story>
</Preview>
