# Breadcrumb

With MDX we can define a story for Breadcrumb right in the middle of our
markdown documentation.

## Default

<Preview>
  <Story name="Simple">
    {() => {
      const items = [
        { to: '/', label: 'Home' },
        { to: '/breadcrums', label: 'Breadcrumb' }
      ];
      return <Breadcrumb items={items} />
    }}
  </Story>
</Preview>

## With Knobs

<Preview>
  <Story name="Knobs">
    <Breadcrumb
      items={[
        { to: '/', label: text('Label One', 'Home') },
        { to: '/breadcrums', label: text('Label Two', 'Breadcrumb') },
      ]}
    />
  </Story>
</Preview>
